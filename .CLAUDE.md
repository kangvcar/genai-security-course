# Jupyter 实验优化要求与偏好

## 总体目标
- 实验用于帮助学生通过动手实践更好理解理论内容。
- 实验为学生自主完成，需提供充分说明、指导与解释，循序渐进。
- 难度控制在简单到中等，面向具备基础 Python 编程与简单机器学习概念的大专高校学生。
- 自学友好：中文详细注释，重点解释“为什么”而非“怎么做”。

## 平台与环境
- 实验平台：腾讯 Cloud Studio（https://cloudstudio.net/），提供免费 NVIDIA T4 GPU（16GB 显存），支持 Jupyter Notebook，国内访问速度快，模型下载稳定。
- 尽量减少不稳定依赖。

## 代码与题目设计约束
- 每个实验总代码量 ≤ 40 行（不含说明性文字）。
- 学生填空 ≤ 5 处；每处 1-2 行。
- 每个填空必须提供【提示】与【参考答案】。
