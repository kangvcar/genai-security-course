- 🛡️ **攻防双视角**：既学习攻击技术，也掌握防御策略
- 📚 **系统化学习路径**：从基础到进阶，循序渐进
</Callout>

1. **按顺序学习**：建议从模块一开始，按顺序完成所有模块
2. **动手实践**：每个模块的配套实验是学习的重要组成部分
3. **完成思考题**：每章末尾的思考题帮助巩固所学知识
4. **持续探索**：关注 AI 安全领域的最新研究和事件
