## 实验目标

1. 解析结构化的模型卡 JSON 数据
2. 编写安全审计检查函数（训练数据透明度、风险披露、许可证等）
3. 对多个模拟模型卡执行自动化审计
4. 生成安全审计报告并给出风险评级

## 实验环境

- 腾讯云 Cloud Studio（NVIDIA T4 GPU）
- Python（本实验不需要 GPU，纯 Python 编程）

<Callout title="实验特点" type="info">
本实验不需要加载 AI 模型，是一个纯 Python 编程练习。通过编写审计脚本来检查模型卡中的安全相关信息，培养系统性思考安全问题的能力。
